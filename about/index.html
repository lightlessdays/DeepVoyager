DeepVoyager is a Large Language Model, created in India.
DeepVoyager is an Open Source, Large Language Model, crafted with loads of love in India.
DeepVoyager can access the internet.
Unlike other LLMs, including ChatGPT and Bard, DeepVoyager is not pre trained on loads of pre defined data. Instead, one each search DeepVoyager uses Omega Search Engine's results to derive optimal result.
DeepVoyager does not track you or your information.
Unlike other LLMs, DeepVoyager is trustworthy. It does not track your information. Don't believe us? Well... all our code is available for the public on GitHub. You can read the code yourself to find out.
DeepVoyager is truly free and ads free.
DeepVoyager is free and adless. But what if we place ads in the future? Well... we simply cannot. Because this project is open source. This implies that even if we place ads, you can create your own ads free copy from our code.
